# MegaBonk Clone Project

MegaBonk를 모티브로 제작한 **로그라이크 액션 슈팅 게임**입니다.

언리얼 엔진 5와 C++를 활용하여 로그라이크 핵심 로직 구현에 중점을 두었습니다.
# 팀원 소개
  - 김성현(https://github.com/SUNMUNG)
    플레이어 이동, 스탯, 무기 관리 시스템, 충돌 및 피격 판정, 오브젝트 풀 구현
  - 김민준(https://github.com/zangsa06)
    몬스터 AI, 보스 몬스터 패턴, 몬스터 스포너 구현
  - 김수진(https://github.com/kdjh2002)
    메인 HUD 및 UI 매니저, (아이템, 상점, 인벤토리, 상호작용 오브젝트) UI 연동, 사운드 시스템 
  - 유성민(https://github.com/YouSungMin)
    각종 데이터 테이블 구축, (1회성 아이템, 패시브 아이템) 효과, 인벤토리 및 레벨 업 시 보상 시스템, 상호작용 오브젝트 구현

# 게임 특징
  - 웨이브 및 추적 AI 시스템: 플레이어를 인식하고 추적하는 몬스터가 웨이브 형태로 끊임없이 스폰됩니다.

  - 재화 및 성장 루프: 몬스터 처치 시 골드와 경험치를 획득하여 실시간으로 캐릭터를 성장시킬 수 있습니다.

  - 로그라이크 성장: 레벨 업 시 무작위로 등장하는 무기/비전서(패시브) 선택지를 통해 매 판 달라지는 전략적 빌드를 구성합니다.

  - 상호작용 오브젝트: 맵 곳곳에 배치된 오브젝트를 통해 버프, 회복 등 전투에 유용한 효과를 획득합니다.

  - 보스전 및 클리어: 성장을 통해 몬스터 웨이브를 생존하고, 최종 보스를 소환 및 처치하여 스테이지를 클리어하는 것이 목표입니다.
